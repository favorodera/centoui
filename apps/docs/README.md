## Development

From the monorepo root:

```bash
# Start the preview app on http://localhost:5173
pnpm --filter preview dev
```

Or from this directory:

```bash
pnpm dev
```

## Build

```bash
# Production build
pnpm build

# Preview the production build locally
pnpm preview
```

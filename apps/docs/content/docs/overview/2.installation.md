---
title: Installation
description: Get started with CentoUI by installing the package and configuring it in your Vue 3 or Nuxt project.
---

CentoUI supports Vue 3 and Nuxt. Because components are copy-pasted into your project, the "installation" is a one-time CLI setup followed by adding the components you need.

## Prerequisites

- Node.js v22 or later
- pnpm v11 or later

## Initialize a Project

Run the init command in an existing Vue or Nuxt project:

```bash [Terminal]
npx centoui init
```

This creates a `centoui.config.ts` file and a theme CSS entry point in your project.

## Add Components

After initialization, add individual components:

```bash [Terminal]
npx centoui add button input
```

The CLI copies the component source files into your project and installs any required dependencies.

## Nuxt Module

For Nuxt projects, you can skip the CLI and use the official module:

```bash [Terminal]
pnpm add centoui-nuxt
```

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['centoui-nuxt']
})
```

The module auto-registers components and respects the optional prefix configuration.

## Verify Installation

Start your dev server and confirm the components render:

```bash [Terminal]
pnpm dev
```

If you used the Nuxt module, components are available immediately. If you used the CLI, import them from the path where they were copied.

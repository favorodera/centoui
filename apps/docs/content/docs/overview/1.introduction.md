CentoUI is a **copy-paste component library** for Vue 3. You run a CLI command, the component source lands in your project, and you own every line. Styling is handled by a single CSS theme file powered by Tailwind CSS v4, so changing your entire look is one file edit away.

## Why copy-paste?

Most component libraries require you to import from `node_modules`. That works until you need to tweak one prop, fix an edge case, or understand why a component behaves a certain way. With CentoUI, the source lives in your repo. You read it, debug it, and customize it like any other file in your project.

## How it works

1. Run `centoui init` in your project.
2. Add components with `centoui add button input`.
3. The CLI copies the `.vue` files into your project.
4. Import and use them like any local component.

## What you get

- Accessible components built on Reka UI primitives.
- Variant-driven styling via Tailwind Variants.
- A single CSS theme file that controls every color token.
- TypeScript types for every component prop.

## Who is it for?

Teams and individuals who want the speed of a component library without the lock-in of npm imports.
